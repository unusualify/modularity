# `Create Vue Input`

> Create Vue Input Component.

## Command Information

- **Signature:** `modularity:create:vue:input <name>`
- **Category:** Generators


## Examples

### With Arguments

```bash
php artisan modularity:create:vue:input NAME
```


`modularity:create:vue:input`
-----------------------------

Create Vue Input Component.

### Usage

* `modularity:create:vue:input <name>`
* `mod:c:vue:input`

Create Vue Input Component.

### Arguments

#### `name`

The name of the component to be created.

* Is required: yes
* Is array: no
* Default: `NULL`
