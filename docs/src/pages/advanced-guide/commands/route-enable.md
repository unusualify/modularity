# `Route Enable`

> Enable the specified module route.

## Command Information

- **Signature:** `modularity:route:enable <module> <route>`
- **Category:** Other


## Examples

### With Arguments

```bash
php artisan modularity:route:enable MODULE ROUTE
```


`modularity:route:enable`
-------------------------

Enable the specified module route.

### Usage

* `modularity:route:enable <module> <route>`

Enable the specified module route.

### Arguments

#### `module`

Module name.

* Is required: yes
* Is array: no
* Default: `NULL`

#### `route`

Route name.

* Is required: yes
* Is array: no
* Default: `NULL`
