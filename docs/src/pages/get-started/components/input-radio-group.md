# Radio Group <Badge type="tip" text="^0.9.2" />

The `v-input-radio-group` component presents radio button selectable wrapper.

## Usage
``` php
  [
    ...,
    'type' => 'radio-group', // type name
    'name' => '_radio-group',
    'itemValue' => 'id',
    'itemTitle' => 'name',
    'items' => [
        [
            'id' => 1,
            'name' => 'Title 1',
        ],
        [
            'id' => 2,
            'name' => 'Title 2',
        ]
    ],
  ],
```

> [!IMPORTANT]
> This component was introduced in [v0.9.2]
